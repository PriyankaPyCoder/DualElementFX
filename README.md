# DualElementFX

## 📖 Overview

DualElementFX is a real-time interactive AR experience built using hand tracking and dynamic particle systems. It allows users to control elemental visual effects (Fire, Ice, Lightning, Red/Blue energy) using hand gestures.

The system uses webcam input to detect hand movements and generates cinematic visual effects with dynamic lighting and physics-based particles.

---

## 🚀 Features

* Real-time hand tracking using MediaPipe
* Gesture-based interaction (open hand, pinch detection)
* Multiple power modes:

  * 🔥 Fire & ❄️ Ice
  * ⚡ Lightning
  * 🔴 Red & 🔵 Blue energy
* Particle system with physics (velocity, decay, glow)
* Dynamic lighting and cinematic dark mode rendering
* Smooth real-time performance in browser

---

## 🛠️ Technologies Used

* JavaScript (Canvas API)
* MediaPipe Hands
* HTML5 & CSS3
* Particle System Simulation

---

## 🎮 Controls

* Press **1** → Fire & Ice mode
* Press **2** → Lightning mode
* Press **3** → Red & Blue mode

### Hand Gestures

* ✋ Open Hand → Activate energy
* 🤏 Pinch Gesture → Draw / control particles
* Movement → Directional effects
